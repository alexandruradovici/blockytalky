BlockyTalky is a low-cost, extensible, open source, programmable, networked toolkit for tangible creations.

To install the system, run chmod 777 ./build && sudo ./build to install all dependencies.
To start the system, run sudo sh start.sh
To restart the system, run sudo sh restart.sh
